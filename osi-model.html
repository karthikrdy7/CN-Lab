The OSI Model

The OSI (Open Systems Interconnection) model splits everything a network does into 7 stacked layers. No single device implements all seven perfectly — but the model is the shared vocabulary engineers use to talk about "where" a problem lives. Click a layer to learn what it's responsible for.

Watch a message cross the stack

When your computer sends something, it doesn't just fire off your data — each layer wraps it in its own header first. This is encapsulation. The receiving computer then unwraps those headers one by one, from the bottom up, called decapsulation.

Real devices don't always touch all 7 layers — a switch mostly only cares about layer 2, a router mostly cares about layer 3. That's exactly why the model is useful: it tells you which layer to suspect when something breaks.
